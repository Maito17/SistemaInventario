    💡 Sistema de Respaldos

    Los respaldos se guardan en backup/respaldos/. Usa los botones de arriba para crear un nuevo respaldo o también puedes usar la terminal: python manage.py respaldo

    🔐 Cada respaldo incluye verificación de integridad SHA256. 📧 Se envían notificaciones por email si un respaldo automático falla. ⬇ Haz clic en "Descargar" para obtener una copia del archivo.

    🔄 SQL: respaldo nativo PostgreSQL (más rápido, solo PostgreSQL). 📋 JSON Portable: formato universal compatible con cualquier base de datos (ideal para migrar).
